## 🛠️ Tech Stack

This site itself is intentionally simple:

- **HTML5** — single self-contained `index.html`
- **CSS3** — custom properties (variables), flexbox layout, responsive
  breakpoints, print-specific styles for a clean A4 PDF export
- **Vanilla JS** — one line, just to trigger the browser's print dialog

No build tools, no dependencies, no frameworks — it just works anywhere a
browser can open an HTML file.

## ✨ Features

- Responsive layout — looks right on desktop, tablet, and mobile
- One-click **Download as PDF** button that fits perfectly on a single A4 page
- Fast load — no JS frameworks, no build step
- Easy to re-skin — colors and fonts are CSS variables at the top of the file
